# 电机零点 / 限位 / 方向整理

## 说明

- 本表按当前代码实际生效逻辑整理，不只看配置名。
- “零点”优先写当前控制链路真正使用的零点。
- “限位”优先写当前软件实际生效的限位；如果只是单圈归一化、没有额外软限位，也会单独说明。
- “方向”拆成 3 类：
  - `RC方向`：遥控/上位机输入为正时，目标是否按正方向变化。
  - `映射方向`：外部 raw 映射到电机 raw 时，正向是否同向。
  - `输出方向`：最终下发电流/力矩时是否乘了 `-1`。

## 云台补充

- `GIMBAL_DM_J4/J5/J6/J7_ZERO_SHIFT_RAD` 目前都只是 legacy manual trim 注释，当前权威零点应以 `CONF_Gimbal_Zero.h` 里的 `ABS_ZERO_RAW / ABS_ZERO_RAD` 为准。
- DM 上电回零开启：
  - J4 `true`
  - J5 `true`
  - J6 `true`
  - J7 `false`
  - J8 `true`，`3000 ms`
  - J9 `true`，`3000 ms`
- LK 上电回零开启：
  - J1 `true`，`3000 ms`
  - J2/J3 当前未单独开启上电回零
- `J1/J2/J3` 的 `POS_DIR_INVERT` 常量都为 `false`，但当前执行链路没有直接使用这 3 个常量，实际方向应以 `RC_SIGN` 和 raw 映射表为准。
